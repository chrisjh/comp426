Welcome To Level

Level is a cutting-edge social media site that will level the playing field for free speech online.
Level places thought over person, taking away bandwagons and blind followers.

How It Works

Users create a profile with basic information (i.e. Name, Location, Organizations, Bio, and Pictures)
This information is initially hidden from other Users.
After creating the User Profile, Users will then be able to do three things:

1) Post a thought
2) Comment on a thought
3) Favorite/Star a thought
4) Follow another User

After a User chooses to follow a thought, the poster's profile will become visible to the follower.

The Basics

Users:
- User ID's (Anonymity)
- Username (Revealed by a follow)
- User ID is mapped to username and is unchanging

Thoughts/Posts:
- Tags
- Associated Users
- Comments
- Content
- Titles
- Starred/Favorite Option
- Related Thoughts*
- Related Tags*

Profile:
- Name
- Username
- User ID
- Tags
- Bio
- Location
- Picture
- Direct Messaging*

* - denotes challenging features that may not be implemented due to time constraints.
